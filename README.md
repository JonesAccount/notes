# Менеджер заметок

Простая программа для работы с личными заметками.  

## Возможности
- Добавление новых заметок  
- Удаление существующих заметок  
- Просмотр всех заметок  

Все данные хранятся с помощью текстовых файлов без использования внешних баз данных.  
Программа подходит для лёгкого и быстрого ведения заметок на компьютере.
